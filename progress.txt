## Codebase Patterns
- Backend structure: backend/app/ with main.py, config.py
- Settings use pydantic-settings BaseSettings with VIGILANTE_ env prefix
- Python venv at backend/.venv
- CORS configured for http://localhost:3000
- All API routes prefixed with /api/
- Use mypy --ignore-missing-imports for typecheck
